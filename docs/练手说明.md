# 人格化对话模型练手包

这个仓库对应一个可直接练手的公开复现题目：

**题目：基于 PersonaChat / Synthetic-Persona-Chat 的人格化对话生成模型复现**

## 文件说明

- `docs/人格化对话模型练手方案.md`：可直接交作业或再加工的中文方案
- `scripts/train_persona_easy.py`：训练脚本，默认使用 `facebook/blenderbot_small-90M`
- `experiments/logs/hf_jobs_training_attempt_log.txt`：真实训练尝试日志
- `app.py`：本地 Web 可视化训练入口
- `launch_windows.bat`：Windows 一键启动脚本

## 推荐资源

### 数据集
- PersonaChat（truecased 版本）  
  https://huggingface.co/datasets/bavard/personachat_truecased

- Synthetic-Persona-Chat  
  https://huggingface.co/datasets/google/Synthetic-Persona-Chat

### 模型
- BlenderBot Small 90M  
  https://huggingface.co/facebook/blenderbot_small-90M

- DialoGPT-small  
  https://huggingface.co/microsoft/DialoGPT-small

## 本地推荐启动方式

直接双击：

```bat
launch_windows.bat
```

它会自动：

1. 创建虚拟环境
2. 安装依赖
3. 启动本地 Web UI

## 设计原则

考虑到当前机器为：

- RTX 4060 Laptop GPU，8 GB 显存
- 16 GB 内存
- Windows 11

因此默认采用：

- 小模型优先
- 冒烟测试优先
- 默认参数可直接跑
- 尽量避免用户手动拼接复杂命令
