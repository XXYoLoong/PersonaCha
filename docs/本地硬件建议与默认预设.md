# 本地硬件建议与默认预设

## 当前已知硬件

根据项目记录，当前可调用设备大致为：

- GPU：NVIDIA GeForce RTX 4060 Laptop GPU
- 显存：8 GB
- 内存：16 GB
- CPU：AMD Ryzen 9 7945HX
- 系统：Windows 11

## 为什么默认走轻量预设

这套硬件已经足够完成小模型 persona 对话训练，但不适合一上来就用超大模型或高 batch size。
因此仓库默认采用 **BlenderBot Small 90M + 5k 样本 + 1 epoch** 的稳妥设置，优先保证：

- 能跑通
- 不容易爆显存
- 日志可观察
- 参数容易理解

## 默认推荐预设

- 模型：`facebook/blenderbot_small-90M`
- 数据集：`bavard/personachat_truecased`
- 训练样本：`5000`
- 验证样本：`500`
- 训练轮数：`1`
- 单卡 batch size：`4`
- 梯度累积：`4`

## 推荐使用顺序

1. 先打开 `launch_windows.bat`
2. 在 Web UI 中点击“应用推荐预设”
3. 先勾选“仅预处理自检”
4. 确认数据和日志没问题后，再取消勾选并开始正式训练

## 不建议的做法

- 一开始就把样本量拉满
- 同时开很多后台程序占用显存
- 直接切换到更大的生成模型而不做冒烟测试
